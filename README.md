# MisterNikto
Метот как разблокировать ютуб, роблокс и другие
1.При скачивании вам нужно вытащить содержимое и открыть services.bat
2.Введите в нужном (последнем) поле 1
3.Введите любое число (советуется 11)
4.Когда написано The operation is successful, то можно открывать ваши сайты, игры
5.Если у вас не работает, то тогда возвращаемя к шагам после шага 1, но вместо введения цифры 1 нужно ввести цифру 2 а после удаления цифру 1 и тд
